landplanner by wboykinm

A Module for editing a PostGIS database from a Client Webmap and back again.

This project is designed specifically to allow ranchers to plan and manage their land use with an eye toward soil carbon conservation and fertility preservation

Dependencies

postgresql-8.4+, postgis-1.5+, python-2.7+, django-1.3, geodjango-1.0, openlayers-2.1, vectorformats,

Install

git clone https://github.com/wboykinm/landplanner
